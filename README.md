Overview
This presentation explores how I went from being curious about AI to becoming an active user who leverages it across multiple aspects of work and life. It's designed to inspire others to explore new ways of using AI and share practical examples from real-world use cases.
What's Covered
My AI Journey

Why I took so long to start using AI (even after completing my degree)
The catalyst that got me started
Evolution from basic usage to advanced applications

Practical AI Applications Demonstrated

Career Development: Cover letter refinement and improvement
Learning & Certification: C# certification practice and preparation
Code Quality: Bug fixing, identifying code smells, and refactoring
Development: "Vibe coding" and component selection
Problem Solving: YouTube appeal writing
Life Planning: The "Eureka Moment" - prioritization and focus

The Eureka Moment
A breakthrough conversation that helped me:

Stop second-guessing and commit to goals
Prioritize competing interests (career, projects, leisure, research)
Create actionable pathways and to-do lists
Build a framework for productivity and focus

Key Takeaways

AI isn't just for coding - it's a versatile tool for problem-solving
Starting small with AI can lead to transformative workflows
AI can help with clarity, focus, and overcoming procrastination
Different use cases: bug fixing, code review, writing, planning, and more

Presentation Goals
